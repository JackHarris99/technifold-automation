# Spine-Creaser Technology

> Single spine crease for saddle stitchers - eliminates cover cracking on magazines and booklets

## What It Does

Creates a single, precise crease on the spine of magazine and booklet covers **inline** on saddle stitching machines. The crease is applied after the cover feeder and before stitching, preventing fibre cracking when the finished product is opened.

## Machine Type

**Saddle Stitchers** - sits in the cover feeder section

## Installation Notes

### General Requirements
- Electrical power to machine must be OFF
- Experienced technician recommended
- Allen wrenches required
- Installation typically a few hours

### Duplo iSaddle Specifics
- Remove existing metal scoring unit
- Install male component on upper shaft
- Install female component on lower shaft
- Align male rib with female channel
- Test with 250-300gsm stock for U-shaped crease

### Horizon StitchLiner Specifics
- **CRITICAL**: Requires upper and lower guide plate modifications
- Cut-outs must be extended by minimum 5mm
- Edges must be rounded and filed smooth
- Modifications should only be made by competent technician

### Muller Cover Feeder Specifics
- Special matrix designed to stretch and snap into feeder drum's existing scoring groove
- Remove existing scoring arm
- Apply creasing matrix to drum
- Install new double-wheel arm
